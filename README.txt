THIS IS A SIMPLE PROJECT THAT UTILIZE HTML, CSS, JACASCRIPT AND MYSQL FOR IMPLEMENTING THE GYM MANAGEMENT SYSTEM WHICH MANAGE THE USERS OR MEMBERS THEIR GYM MEMBERSHIP. ITS MAIN GOAL IS TO SELL UTILITIES AND PRODUCTS RATHER THAN USER MEMBERSHIP. 

PLEASE CONSIDER THE INSTRUCTIONS BELOW BEFORE TESTING THE SYSTEM.

-NAVIGATE THROUGH FONTS FOLDER AND SELECT THE FOLDERS THAT CONTAINS THE FONTS.
-RIGHT CLICK THROUGH EACH OPENTYPE FORMAT THEN SELECT INSTALL.
-AFTER SUCCESSFULLY INSTALLING THE FONTS YO CAN NOW RUN AND TEST THE FILE.
